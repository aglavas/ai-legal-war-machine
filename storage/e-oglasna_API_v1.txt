e-Oglasna API
Ispod vršne adrese API-a e-Oglasne https://e-oglasna.pravosudje.hr/api/v1/* nalaze se servisi koji
prihvaćaju pozive iz vanjskih sustava i vraćaju podatke o oglasima objavljenima na oglasnoj ploči

Dokumentacija za API krajnje točke
Svi URL-ovi su relativni prema: https://e-oglasna.pravosudje.hr
HTTP zahtjev
GET /api/v1/notice
GET /api/v1/notice/{uuid}

Opis
Dohvaća sve oglase, rezultati pretrage odgovaraju rezultatima pretrage
na stranici https://e-oglasna.pravosudje.hr/objave
Dohvaća oglas po UUID-u
Dohvaća oglase sa suda, rezultati pretrage odgovaraju rezultatima

GET /api/v1/court-notice

pretrage na stranici https://e-oglasna.pravosudje.hr/objavesudova

GET /api/v1/court-

Dohvaća oglas po UUID-u, detalji oglasa sadrže i atribute specifične za

notice/{uuid}

oglas sa suda

GET /api/v1/courtnotice/legal-personbankruptcy
GET /api/v1/courtnotice/legal-personbankruptcy/{uuid}
GET /api/v1/courtnotice/natural-personbankruptcy
GET /api/v1/courtnotice/natural-personbankruptcy/{uuid}
GET /api/v1/institutionnotice

Dohvaća oglase sa suda, rezultati pretrage odgovaraju rezultatima
pretrage na stranici https://e-oglasna.pravosudje.hr/stecajevi

Dohvaća oglas po UUID-u, detalji oglasa sadrže i atribute specifične za

Rate limiting
API ima rate limiting (https://en.wikipedia.org/wiki/Rate_limiting) i koristi token bucket algoritam
(https://en.wikipedia.org/wiki/Token_bucket) u implementaciji rate limiting pravila.
Pravila kod pozivanja metoda servisa:
Maksimalno 1000 poziva u periodu od 1 sata
Maksimalno 5 poziva unutar vremenskog prozora od 5 sekundi
U slučaju da je dosegnut limit u pozivanju servisa, servis odgovara HTTP statusom 429 - TOO MANY
REQUESTS, a u HTTP headeru X-Rate-Limit-Retry-After-Milliseconds nalazi se podatak o vremenu koje
treba proteći prije slijedećeg prihvaćanja novog HTTP zahtjeva.

Kategorije
Za pojedine korisnike uvedene su kategorije Rate limiting postavki. Ti korisnici određeni su fiksnim IP
adresama
Kategorija
ADVANCED

Postavke
maksimalno 5000 poziva na sat, maksimalno 15 poziva unutar vremenskog prozora od 5
sekundi

Primjeri
Pretraga oglasa

curl 'https://e-oglasna.pravosudje.hr/api/v1/notice?text=abc'

Dohvat šifrarnika sudova
Dohvat šifrarnika sudova pozivom servisa korištenjem naredbe curl:

curl 'https://e-oglasna.pravosudje.hr/api/v1/court/all'

Generiranje Markdown dokumentacije iz YAML datoteke

docker run --rm -v $(pwd):/local openapitools/openapi-generator-cli generate -i
/local/e-oglasna_API_v1.yaml -g markdown -o /local
